# Todo App

요일별 할 일 관리 앱

## 기능

- 📅 요일별 할 일 관리 (월/화/수/목/금/휴일)
- 📊 주간 달성률 추적
- 💾 전체 저장/불러오기
- 📋 요일별 복사/붙여넣기
- 📊 엑셀 Export
- 🌙 다크 모드

## 사용 방법

1. `index.html` 파일을 브라우저에서 열기
2. 원하는 요일 탭 선택
3. 할 일 추가 및 관리

## 키보드 단축키

- `Ctrl/Cmd + K`: 입력 필드 포커스
- `Ctrl/Cmd + D`: 다크 모드 토글
- `Ctrl/Cmd + S`: 전체 저장
- `Ctrl/Cmd + O`: 전체 불러오기
- `Ctrl/Cmd + E`: 엑셀 Export
- `Ctrl/Cmd + Shift + C`: 현재 요일 복사
- `Ctrl/Cmd + Shift + V`: 다른 요일에 붙여넣기

## 기술 스택

- HTML5
- CSS3 (Vanilla CSS)
- JavaScript (ES6+)
- SheetJS (엑셀 Export)
